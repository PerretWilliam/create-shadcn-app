# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-11-09

### Added

- Initial release
- Support for React + TypeScript + SWC via Vite
- Automatic Tailwind CSS v4 setup with Vite plugin
- ShadCN UI integration
- TypeScript configuration with path aliases (`@/*`)
- Multiple package manager support (npm, pnpm, yarn, bun)
- Smart directory conflict handling
- Clean project structure generation
- Comprehensive README documentation
